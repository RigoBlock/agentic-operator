Delegate to Agent

1 Agent Wallet

A dedicated wallet will be created for the agent to execute transactions on your behalf. Its keys are encrypted and stored securely.

2 Delegated Functions

You’ll send a transaction to your vault calling updateDelegation() which grants the agent permission to call specific trading functions:

  • Uniswap execute / modifyLiquidities
  • 0x Aggregator execute
  • GMX v2 create / update / cancel orders, claim fees

3 Confirm Transaction

Sign and send the delegation transaction from your wallet. You can revoke at any time by calling revokeAllDelegations(agentAddress).
